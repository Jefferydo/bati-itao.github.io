Déterminer l’objet de la saisie (overview)

FRENCH: re-arranged sentences below

When a form input field collects personal information such as the user’s name, phone number, email and street addresses, you must programmatically define its semantics with the autocomplete attribute. Assistive technologies leverage the autocomplete attribute value to automatically complete forms, drawing on past user input stored in the browser. This saves the user from having to type the information. Assistive technologies also leverage the autocomplete value to pair input fields with icons, for users who prefer using images for communication.

The attribute value takes one of 53 potential values field names from WCAG’s list of Input Purposes for User Interface Components. The values are granual The field names are either broad or narrow. For instance, regarding the user’s name, it’s possible to identify an input’s purpose as full name, given name, family name, or nickname. Lorsqu’un champ de saisie de formulaire recueille des renseignements personnels comme le nom de l’utilisateur, son numéro de téléphone, son adresse électronique et son adresse municipale, vous devez définir sa sémantique par programmation au moyen de l’attribut autocomplete. L’attribut prend l’une des 53 valeurs potentielles parmi la liste Input Purposes for User Interface Components des WCAG. Les valeurs sont granulaires; par exemple, en ce qui concerne le nom de l’utilisateur, il est possible d’identifier la fonction d’une saisie comme le nom complet, le prénom, le nom de famille ou le surnom.

Les technologies d’assistance tirent parti de la valeur d’attribut autocomplet pour remplir automatiquement les formulaires, en s’appuyant sur les données de saisie antérieures stockées dans le navigateur. Cela évite à l’utilisateur d’avoir à taper l’information. Les technologies d’assistance tirent également parti de la valeur autocomplete pour jumeler les champs de saisie avec des icônes, pour les utilisateurs qui préfèrent utiliser des images à des fins de communication.

Use autocomplete on <input> elements that take a text or numeric value as input, <textarea> elements, <select> elements, and <form> elements. The HTML5 specification lists the appropriate elements to match with each autocomplete field name.

<form method="post" action="step1">  
  <label for="fname">Prénom</label>
  <input id="fname" type="text" autocomplete="given-name">  

  <label for="lname">Nom de famille</label>  
  <input id="lname" type="text" autocomplete="family-name">   
  
  <label for="jobtitle">Titre du poste</label>  
  <input type="text" id="jobtitle" autocomplete="organization-title">
  
  <label for="org">Organisation</label>
  <input type="text" id="org" autocomplete="organization">
  
  <fieldset>
    <legend>Shipping address</legend>
    
    <label for="ship-st">Street</label>
    <input type="text" id="ship-st" autocomplete="shipping street-address">
    
    <label for="ship-city">City</label>
    <input type="text" id="ship-city" autocomplete="shipping address-level2">
    
    <label for="ship-prov">Province</label>
    <input type="text" id="ship-prov" autocomplete="shipping address-level1">
    
    <label for="ship-mobile-phone">Mobile phone for delivery</label>
    <input type="tel" id="ship-mobile-phone" autocomplete="shipping mobile tel-national">
    
    <label for="ship-email">Email</label>
    <input type="email" id="ship-email" autocomplete="shipping email">
  </fieldset>
  
  <fieldset>
    <legend>Billing address</legend>
    
    <label for="bill-st">Street</label>
    <input type="text" id="bill-st" autocomplete="billing street-address">
    
    <label for="bill-city">City</label>
    <input type="text" id="bill-city" autocomplete="billing address-level2">
    
    <label for="bill-prov">Province</label>
    <input type="text" id="bill-prov" autocomplete="billing address-level1">
    
    <label for="bill-phone">Phone</label>
    <input type="tel" id="bill-phone" autocomplete="billing tel-national">
    
    <label for="bill-email">Email</label>
    <input type="email" id="bill-email" autocomplete="billing email">
  </fieldset>
